Projects that I made in Delfi Technologies

Some of the projects I have been working on, in the last 6 months!


  • Printer Rollout project
    This project was the first real assingment I got. I had to be the project manager, an make descriptions, flowchart etc. of an application for our technicians. The application is used for documentation of the installation of 1000 new printers in all Netto DK stores. They can add pictures, IP-adresses, Gateways, descriptions etc.
  • API cXML order data conversion
    An API that recives orders from one of our big custmores, they are using an external site to place there oders to us, an we recive the order in an cXML format. Then my program converts the data to an .xml file of my own creation, that is uploaded to our FTP-server, where we can import the order to Microsoft Navision!
  • Convert external stock data
    My biggest project yet, spend almost 2 months writing this. Delfi has two external suppliers, and we have to extract to files pr supplier. One with the stocklevels, witch is update every hour, and one with prices, witch is update once a day! My application collect that data, one supplier with FTP connection, and one with direct download link. Then I convert the data from .xml, .xls, .dat, .xml, to a new xml-file(design by my and our team), then uploads it to our own FTP-server. Then our Microsoft Navision program reads the files, an updates all of our systems.
